# draw-cardioid-python
Draws a cardioid 
(Install pygame according to requirements.txt) ("pip install -r requirements.txt" command in console probably)

‚Åç Play around with main.py lines 28, 33, 35!!!

Method:
1. Calculate evenly spaced points along the circumference of circle (maybe 20) (numbered counter-clockwise)
2. Draw lines connecting point number i with point number i*2 (1-2,2-4,3-6,4-8...)
3. Cardioid appears!!
